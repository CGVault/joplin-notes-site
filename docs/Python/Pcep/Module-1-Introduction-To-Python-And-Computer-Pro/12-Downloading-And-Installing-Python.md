
## Downloading Python

Linux users - most likely have python installed already as much of the OS  features depend on python. May need to upgrade to the latest version.

Windows users - Download the installer from the website, check the ADD TO PATH checkbox.

IDLE -  to develop python scripts you need a Integrated Development and Learning Environment. Python comes with one but there are better options.

Source code - python scripts are installed in .py files, so either create files as .py first or rely on your IDLE's saving feature.
